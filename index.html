Instructions:
  1. Set browser as the OS default browser
  2. Enable feature flag #enable-desktop-pwas-url-handling
  3. Install PWA from current page
  4. (Optional) Disable feature flag to simulate Origin Trial behavior
  5. Activate test URLs from the OS (eg. in Windows using the run dialog)

Troubleshooting instructions:
  1. Visit about://web-app-internals to check that "url_handlers" in manifest was parsed and applied
  2. Visit about://local-state to check that web-app-origin-association data has been validated and saved to "url_handler_info"
